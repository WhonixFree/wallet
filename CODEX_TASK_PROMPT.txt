Build a demo-first frontend-only version of this local Windows desktop crypto portfolio tracker.

Goal:
make the app look fully working, polished, and production-ready in recorded demos, promo videos, and walkthroughs, even though backend functionality is not implemented yet.

Important constraints:

keep the product as a local Windows desktop app
keep Electron as the desktop shell
keep React as the renderer/UI layer
do not implement Go backend in this phase
do not implement SQLite in this phase
do not implement real HTTP/API calls
do not implement real blockchain, explorer, CoinGecko, or sync integrations
do not implement real file persistence for notes
do not implement real notifications
do not implement real AI
do not add auth, cloud accounts, transaction signing, swaps, bridges, or wallet-connect style features
keep the app read-only

Primary objective:
the app should be convincing enough that a user can record a marketing/demo video and it looks like a real finished product.

Build requirements:

polished Electron app shell
React routing and navigation
desktop-focused dark UI
sidebar and page header layout
realistic seeded mock data
async mock service layer
believable loading, empty, success, and error states
smooth transitions, hover states, modals, inline feedback, and status badges
no dead buttons
no TODO placeholders
no visibly unfinished screens

Implement these surfaces:

Dashboard
Wallets list
Wallet details
Groups list
Group details
Alerts
Calendar / reminders
Global search
Notes editor / preview UI
Transfers UI
Import preview UI
Demo assistant chat
